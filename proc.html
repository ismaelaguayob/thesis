Descarga de datos

Utilizamos paquete de python “bcn_scraper” de desarrollo propio para desacargar de forma programática los datos tramitación de la ley 21.735. El paquete desambigüa número de ley o de boletín, descargando automáticamente los documentos de tramitaciones, apilando los datos en un dataframe, y limpiando el xml para obtener el texto plano de cada trámite.

from bcn_scraper.wrapper import historia_dataframe_from_ley_o_boletin

df = historia_dataframe_from_ley_o_boletin(numero_ley="21735", clean_text=True)

df.to_csv("data/datos_bcn_raw.csv", index=False, sep = ";")
